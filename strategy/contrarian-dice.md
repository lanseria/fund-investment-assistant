### 🎲 逆向骰子型（反向操作 + 随机漫步盲注）

**🎯 决策核心原则：**
"市场是集体情绪的放大器，而情绪在绝大多数时候是错的。"本策略主动站在人群对面——**人弃我取、人取我与**：当所有人贪婪（红盘狂欢）时我卖出，当所有人恐惧（绿盘暴跌）时我买入。但纯粹的反向操作容易被聪明资金反向收割，因此引入 **"骰子掷注"机制**：在反向方向确定后，用伪随机信号（基金代码尾数、当日日期、距上一笔交易的天数）决定"是否出手、出多少手"，让对手永远摸不清你的节奏。

**🎲 五面骰决策引擎（本策略的灵魂）：**
每个交易日开盘前摇一次，结果当日不可推翻重摇：

| 骰子           | 取值方式                                             | 含义                                       |
| -------------- | ---------------------------------------------------- | ------------------------------------------ |
| **代码尾数骰** | 持仓 / Watchlist 基金代码最后一位数字（0-9）         | 奇数 → 倾向卖出，偶数 → 倾向买入           |
| **日期骰**     | (当日 dd + 星期几 1~7) 对 3 取余                     | 0 = 静默不出手，1 = 半仓试探，2 = 常规出击 |
| **间隔骰**     | 距 `recentTransactions` 最近一笔的天数               | 奇数天 → 执行反向操作，偶数天 → 按兵不动   |
| **板块轮盘骰** | Watchlist 按 `sectorSignal` 排序，取第 (日期尾数) 个 | 无视其评级，直接作为今日候选标的           |
| **金额盲盒骰** | 基础金额 × (基金代码前两位 / 100 + 0.5)              | 输出区间为总资金 2% ~ 8% 的随机金额        |

**📥 买入操作逻辑（市场越跌越买，金额随机加码）：**
（仅动用"逆向作战资金"，核心防卫现金全程隔离）

1. **常规反向买**
   - **条件**：标的 `percentageChange` < -1%（恐慌初现），且"日期骰" ≠ 0，且"间隔骰"判定执行。
   - **执行**：按"金额盲盒骰"输出买入，金额区间 **2% ~ 5%**。

2. **极致反向（铁律触发）**
   - **条件**：`bias20` < -8%（深度超卖）+ `signals['rsi']` < 30。
   - **执行**：**无视所有骰子结果，强制买入**。金额固定为总资金的 **6% ~ 8%**，分 2 笔在当日 14:30 与次日开盘打完。

**📤 卖出操作逻辑（市场越涨越卖，时点随机）：**
（卖出回笼资金优先回补"逆向作战资金池"）

1. **常规反向卖**
   - **条件**：标的 `percentageChange` > +1.5%（市场狂欢），且"代码尾数骰"为奇数。
   - **执行**：按"金额盲盒骰"卖出当前可用持仓的 **20% ~ 40%**。

2. **极致反向（铁律触发）**
   - **条件**：`bias20` > +8%（严重超买）+ `signals['rsi']` > 70。
   - **执行**：**无视骰子，强制卖出当前可用持仓的 50%**，让狂欢者接盘。

3. **盲盒止盈**
   - **条件**：`profitRate` > 0 且"间隔骰"判定执行。
   - **执行**：按"金额盲盒骰"随机清仓一部分，制造 **无规律止盈**，防止被反向收割。

**🔀 灵活转换逻辑（反板块共识 + 盲盒调仓）：**

- **逆共识切换（Convert Out）**：当 `sectorSignal` 评级"强烈看涨"（人人看多）时，反而 convert_out 该板块至货基暂避；当评级"看跌"（人人看空）时，反而 convert_in 接货。
- **月度盲盒调仓（Convert In）**：每月最后一个交易日，按"板块轮盘骰"从 Watchlist 随机抽 1 个标的，强制 convert_in 等额切换，打破持仓惯性，避免与共识共振。

**⚠️ 三条不可违背的铁律：**

1. **永远反向**：禁止在 `percentageChange` > +2% 时买入，禁止在 `percentageChange` < -2% 时卖出（除非触发"极致反向"铁律）。
2. **尊重随机**：骰子一旦摇出，当日不可推翻重摇；连续亏损也不可修改下一天的骰子规则——**赌徒的纪律就是没有纪律**。
3. **保留弹药**：始终保留 **30% 现金 / 货基**作为"逆向作战的粮食"，禁止一次性梭哈；子弹打光时转入"静默期"，等日期骰 = 0 的次日再开火。
